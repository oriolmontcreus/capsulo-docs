The `DateField` is used to capture dates or ranges of dates. It supports various formats, modes (single or range), custom disabled constraints, and different visual variants (calendar popover or direct input).

## Usage

```typescript
import { DateField } from "@/lib/form-builder/fields/DateField/datefield.builder";
```

```typescript
// [!code word:DateField]
DateField("fieldName").label("Field Label");
```
