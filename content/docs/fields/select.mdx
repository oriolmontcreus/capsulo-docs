The `Select` field provides a dropdown interface for choosing from a list of options. It supports two modes: a standard select dropdown and a searchable combobox. Advanced features include grouped options, multi-column layouts, virtualization for large lists, and internal links mode for page selection.

## Usage

```typescript
import { Select } from "@/lib/form-builder/fields/Select/select.builder";
```

```typescript
// [!code word:Select]
Select("fieldName")
  .label("Field Label")
  .options([
    { label: "Option 1", value: "opt1" },
    { label: "Option 2", value: "opt2" },
  ]);
```

## Type Definitions

### SelectOption

```typescript
interface SelectOption {
  label: string; // Display text for the option
  value: string; // Value when option is selected
  prefix?: ReactNode; // Optional icon or element before label
  suffix?: ReactNode; // Optional icon or element after label
  disabled?: boolean; // Whether the option is disabled
  description?: string; // Secondary text shown in muted color
}
```

### SelectOptionGroup

```typescript
interface SelectOptionGroup {
  label: string; // Group heading text
  options: Array<SelectOption>; // Options within this group
}
```

### ResponsiveColumns

```typescript
interface ResponsiveColumns {
  base?: number; // Default columns (mobile)
  sm?: number; // Small screens (640px+)
  md?: number; // Medium screens (768px+)
  lg?: number; // Large screens (1024px+)
  xl?: number; // Extra large screens (1280px+)
}
```
