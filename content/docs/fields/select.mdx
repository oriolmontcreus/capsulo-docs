The `Select` field provides a dropdown interface for choosing from a list of options. It supports two modes: a standard select dropdown and a searchable combobox. Advanced features include grouped options, multi-column layouts, virtualization for large lists, and internal links mode for page selection.

## Usage

```typescript
import { Select } from "@/lib/form-builder/fields/Select/select.builder";
```

```typescript
// [!code word:Select]
Select("fieldName")
  .label("Field Label")
  .options([
    { label: "Option 1", value: "opt1" },
    { label: "Option 2", value: "opt2" },
  ]);
```
