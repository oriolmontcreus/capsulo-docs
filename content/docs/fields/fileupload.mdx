The `FileUpload` field enables users to upload and manage files including images, documents, and media. It supports multiple variants, file validation, image optimization, and convenient presets for common use cases like avatars and cover images.

## Usage

```typescript
import { FileUpload } from "@/lib/form-builder/fields/FileUpload/fileUpload.builder";
```

```typescript
// [!code word:FileUpload]
FileUpload("fieldName").label("Field Label");
```

## Convenience methods

The `FileUpload` builder also provides several convenience methods for common configurations:

- **`images()`**: Configures the field for image uploads with WebP conversion and quality settings
- **`documents()`**: Configures the field for document uploads (.pdf, .doc, .docx, .txt, .rtf)
- **`media()`**: Configures the field for all media types (images, videos, audio)
- **`inline()`**: Sets the field to inline variant with single file mode
- **`avatar()`**: Creates an avatar upload field with square aspect ratio and 2MB limit
- **`cover()`**: Creates a cover image upload field with wide aspect ratio and 5MB limit
- **`thumbnail()`**: Creates a thumbnail upload field with square aspect ratio and 1MB limit
