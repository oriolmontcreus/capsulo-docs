The `Tabs` layout allows you to divide complex forms into logical sections accessible via a tabbed interface. This is particularly useful for reducing vertical scroll on long forms.

## Usage

```typescript
import { Tabs, Tab } from "@/lib/form-builder/layouts/Tabs/tabs.builder";
```

```typescript
// [!code word:Tabs]
Tabs()
  .tab("Profile", [Input("name").label("Name")])
  .tab("Settings", [Input("theme").label("Theme")]);
```

## API

The `Tabs` builder provides a fluent API to configure the layout and its contents.

### Tabs Methods

<TypeTable
  type={{
    variant: {
      description: "Sets the visual style of the tabs.",
      type: "'default' | 'vertical'",
      default: "'default'",
    },
    tab: {
      description: "Directly adds a tab with a label and its fields.",
      type: "(label, fields, options?) => this",
    },
    addTab: {
      description: "Adds a tab using a pre-configured TabBuilder instance.",
      type: "(tabBuilder) => this",
    },
    className: {
      description: "Applies custom CSS classes to the tabs container.",
      type: "string",
    },
    hidden: {
      description: "Conditionally hides the tabs layout.",
      type: "boolean | function",
      default: "false",
    },
  }}
/>

```typescript
// [!code word:Tabs]
Tabs()
  .variant("vertical")
  .tab("General", [
    Input("siteName").label("Site Name"),
    Input("description").label("Description"),
  ])
  .tab("Security", [
    Input("password").label("Password"),
    Input("recoveryEmail").label("Recovery Email"),
```

</ComponentPreview>

<ComponentPreview showMargin={false} className="border rounded-lg bg-card my-4">
  <SchemaRenderer schema={TabsWithIconsSchema} />

```typescript
// [!code word:Tabs]
Tabs()
  .addTab(
    Tab("Personal", [Input("name").label("Name")])
      .prefix(<UserIcon className="size-4" />)
  )
  .addTab(
    Tab("Settings", [Input("theme").label("Theme")])
      .prefix(<SettingsIcon className="size-4" />)
      .suffix(<Badge variant="secondary">New</Badge>)
  );
```

</ComponentPreview>
