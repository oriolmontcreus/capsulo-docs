The `Textarea` field is used to capture multi-line text input such as descriptions, comments, or longer form content. It supports various configurations including validation, auto-resizing, fixed dimensions, and prefix/suffix elements.

## Usage

```typescript
import { Textarea } from "@/lib/form-builder/fields/Textarea/textarea.builder";
```

```typescript
// [!code word:Textarea]
Textarea("fieldName").label("Field Label");
```
