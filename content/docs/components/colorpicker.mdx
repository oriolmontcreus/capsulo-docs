The `ColorPicker` field is used to capture color values. It supports hex color values, alpha channel transparency, and preset color options.

## Usage

```typescript
import { ColorPicker } from "@/lib/form-builder/fields/ColorPicker/colorpicker.builder";
```

```typescript
// [!code word:ColorPicker]
ColorPicker("fieldName").label("Field Label");
```

## API

The `ColorPicker` builder provides a fluent API to configure the field.

<TypeTable
  type={{
    label: {
      description: "Sets the label of the color picker field.",
      type: "string",
    },
    description: {
      description: "Adds a helper description below the field.",
      type: "string",
    },
    required: {
      description: "Marks the field as required. Can be conditional.",
      type: "boolean | function",
      default: "true",
    },
    defaultValue: {
      description: "Sets the initial value of the field (hex color string).",
      type: "string",
    },
    showAlpha: {
      description: "Enables alpha channel controls for transparency.",
      type: "boolean",
      default: "false",
    },
    presetColors: {
      description: "Array of preset color swatches to display.",
      type: "string[]",
    },
    showInTable: {
      description: "Shows this field in the collection table view.",
      type: "boolean",
      default: "true",
    },
    hidden: {
      description: "Hides the field conditionally.",
      type: "boolean | function",
      default: "false",
    },
  }}
/>
