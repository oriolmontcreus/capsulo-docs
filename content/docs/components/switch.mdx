The `Switch` field is used to capture boolean values in a toggle format. It's ideal for settings, preferences, conditional fields, and on/off states.

## Usage

```typescript
import { Switch } from "@/lib/form-builder/fields/Switch/switch.builder";
```

```typescript
// [!code word:Switch]
Switch("fieldName").label("Field Label");
```

## API

The `Switch` builder provides a fluent API to configure the field.

<TypeTable
  type={{
    label: {
      description: "Sets the label of the switch field.",
      type: "string",
    },
    description: {
      description: "Adds a helper description below the field.",
      type: "string",
    },
    required: {
      description: "Marks the field as required. Can be conditional.",
      type: "boolean | function",
      default: "false",
    },
    defaultValue: {
      description: "Sets the initial value of the field.",
      type: "boolean",
      default: "false",
    },
    showInTable: {
      description: "Shows this field in the collection table view.",
      type: "boolean",
      default: "true",
    },
    hidden: {
      description: "Hides the field conditionally.",
      type: "boolean | function",
      default: "false",
    },
  }}
/>
