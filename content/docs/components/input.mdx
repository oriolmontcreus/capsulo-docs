The `Input` field is used to capture text, numbers, emails, passwords, and other single-line values. It supports various configurations including validation, prefixes, suffixes, and different input types.

## Usage

```typescript
import { Input } from '@/lib/form-builder/fields/Input/input.builder';

Input('fieldName')
  .label('Field Label')
```

## API Reference

The `Input` builder provides a fluent API to configure the field.

| Method | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `label(value)` | `string` | - | Sets the label of the input field. |
| `description(value)` | `string` | - | Adds a helper description below the field. |
| `placeholder(value)` | `string` | - | Sets the placeholder text. |
| `required(value)` | `boolean` \| `function` | `true` | Marks the field as required. Can be conditional. |
| `defaultValue(value)` | `string` | - | Sets the initial value of the field. |
| `type(value)` | `'text'` \| `'email'` \| `'url'` \| `'password'` \| `'number'` | `'text'` | Sets the HTML input type. |
| `prefix(value)` | `ReactNode` | - | Adds an element (icon or text) to the start of the input. |
| `suffix(value)` | `ReactNode` | - | Adds an element (icon or text) to the end of the input. |
| `minLength(value)` | `number` | - | Sets the minimum number of characters. |
| `maxLength(value)` | `number` | - | Sets the maximum number of characters. |
| `min(value)` | `number` | - | Sets the minimum numeric value (for number inputs). |
| `max(value)` | `number` | - | Sets the maximum numeric value (for number inputs). |
| `step(value)` | `number` | - | Sets the step interval (for number inputs). |
| `allowDecimals(value)` | `boolean` | `true` | Allows decimal numbers (for number inputs). |
| `regex(value)` | `string` \| `RegExp` | - | Validates the input against a regular expression. |
| `translatable(value)` | `boolean` | `true` | Enables translation for this field. |
| `showInTable(value)` | `boolean` | `true` | Shows this field in the collection table view. |
| `hidden(value)` | `boolean` \| `function` | `false` | Hides the field conditionally. |
