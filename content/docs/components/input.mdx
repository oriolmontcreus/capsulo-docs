The `Input` field is used to capture text, numbers, emails, passwords, and other single-line values. It supports various configurations including validation, prefixes, suffixes, and different input types.

## Usage

```typescript
import { Input } from '@/lib/form-builder/fields/Input/input.builder';
```

```typescript
Input('fieldName')
  .label('Field Label')
```

## API Reference
